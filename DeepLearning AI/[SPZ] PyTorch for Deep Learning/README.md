# 🧠 PyTorch for Deep Learning Notebooks

This repository contains a collection of hands-on Jupyter notebooks focused on learning and experimenting with deep learning using **PyTorch**.

The goal is to build a strong intuitive and practical understanding of neural networks by implementing models from scratch, training pipelines, and debugging tools such as profiling and visualization.

This work is a personal reimplementation and extension of concepts covered in the [Pythorch for Deep Learning Professional Certificate](https://learn.deeplearning.ai/specializations/pytorch-for-deep-learning-professional-certificate/information) by Laurence Moroney.
